Built and deployed **ReadWise**: a mood-based book recommendation web app.

What I built:

- Book discovery from Google Books + Open Library
- Full web app with auth, profile management, and REST endpoints
- Production deployment on Render

Tech stack:
Python, Django, Django REST Framework, JavaScript, SQLite/PostgreSQL-ready, Google OAuth, Render
